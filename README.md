# Aunt_fastapi

Auth микросервис (FastAPI)
Стек:  FastAPI, SQLAlchemy, PostgreSQL, Redis, Alembic, Kafka, Docker, Docker-compose
Спроектировать БД и расшарить ментору (dbdiagrams.io)
Регистрация пользователей.
Сброс пароля пользователем
Отправка ссылки для подтверждения регистрации на почту
Подтверждение регистрации пользователя
Возможность повторной отправки ссылки на почту
CRUD Roles, Permissions для админов
JWT-Auth (access, refresh, получение информации о пользователе по токену)
Получение информации о всех пользователях (реализовать пагинацию), обновление своего профиля, удаление своего профиля
Сортировка пользователей по дате создания профиля, ролям, именам
Фильтрация пользователей по ролям
